# CloudMoon InPlay

Cloudmoon InPlay is a simple site that proxys, hides, and loads cloudmoon in a browser using Cloudflare workers, alowing you to effortlessley play Roblox, Fortnight, Call of Duty Mobile, Delta Force, and More in Browser at School!

## Use

> [!IMPORTANT]
> Because off googles authentacation polacys, the google sign in buuton will NOT WORk. you must hit sign in with email and password instead. you will aslo need to register you cloudmoon acount at home with google and set a password in settings.

One signed in, you can click and play games in Cloudmoons libary!
